# URE Brasil - Carteirinha Estudantil Digital

## Sobre o Projeto
Sistema oficial para emissão e gestão de Carteirinhas Estudantis Digitais da URE Brasil.

**URL**: https://ure.vendatto.digital

## Tecnologias

Este projeto utiliza:
- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Supabase

## Desenvolvimento

Pré-requisitos: Node.js & npm.

```sh
# Instalar dependências
npm i

# Iniciar servidor de desenvolvimento
npm run dev
```

## Deploy

O deploy é realizado automaticamente via Vercel ao realizar push na branch main.
